---

### ⚙️ Core Architecture

* Multi-model orchestration (parallel + comparative execution)
* Combination of **offline + online models**
* Dynamic routing based on task complexity
* Output synthesis (models compared → refined → merged)

---

### 🧠 Execution Logic

* A single query is processed by multiple models
* Each model generates a different output
* Outputs are **compared against each other**
* Final response is selected and refined

This eliminates:

* repetitive outputs
* low-quality reasoning
* shallow responses
